### Get All Users:
```sql
SELECT id, name, email, role, created_at, 
       CASE WHEN password_hash IS NOT NULL THEN 'Yes' ELSE 'No' END as has_password,
       CASE WHEN email_verified IS NOT NULL THEN 'Yes' ELSE 'No' END as email_verified
FROM users 
ORDER BY created_at DESC;
```

### Get User Tenant Memberships:
```sql
SELECT u.id, u.name, u.email, u.role as system_role, 
       tm.role as tenant_role, t.name as tenant_name, 
       tm."createdAt" as joined_at
FROM users u
LEFT JOIN "TenantMember" tm ON u.id = tm."userId"
LEFT JOIN "Tenant" t ON tm."tenantId" = t.id
ORDER BY u.created_at DESC;
```
