## Getting Started

### Creating an Account

1. Go to the application URL
2. Click "Sign Up" or "Register"
3. Fill in your details:
   - Email address
   - Password
   - First name & Last name
   - Phone number (optional)
4. Click "Create Account"
5. Verify your email (if required)
6. Log in with your credentials

### Logging In

1. Click "Login" or "Sign In"
2. Enter your email and password
3. Click "Sign In"
4. You'll be redirected to your dashboard

## For Organizers

### Creating an Event

#### Step 1: Navigate to Create Event
1. Log in to your account
2. Go to Dashboard
3. Click "Create Event" or "+ New Event"

#### Step 2: Fill Event Details
1. **Basic Information**:
   - Event Name (required)
   - Description
   - Event Type (Conference, Wedding, Concert, etc.)

2. **Date & Time**:
   - Start Date & Time
   - End Date & Time

3. **Venue Details**:
   - Venue Name
   - Address
   - City
   - State/Province
   - Country

4. **Capacity**:
   - Maximum Attendees
   - Enable seat selection (optional)

5. **Images**:
   - Upload event banner
   - Upload additional images

6. **Settings**:
   - Registration deadline
   - Approval required (Yes/No)
   - Public/Private event

#### Step 3: Save Event
- Click "Save as Draft" to save without publishing
- Click "Publish" to make event live

### Managing Registrations

#### View All Registrations
1. Go to your event dashboard
2. Click "Registrations" tab
3. You'll see a list of all registrations

#### Filter & Search
- Use search bar to find by name/email
- Filter by status: All, Pending, Approved, Rejected
- Filter by type: General, VIP, Exhibitor

#### Approve/Reject Registrations
1. Find the registration
2. Click "Approve" or "Reject" button
3. Add notes (optional)
4. Confirm action

#### Export Registrations
1. Go to Registrations tab
2. Click "Export" button
3. Choose format (CSV, Excel, PDF)
4. Download file

### Creating Floor Plans

#### Step 1: Access Floor Plan Designer
1. Go to your event dashboard
2. Click "Design" or "Floor Plans" tab
3. Click "Create Floor Plan"

#### Step 2: Design Layout
1. **Canvas Setup**:
   - Set canvas size (width x height)
   - Choose background color
   - Enable grid (optional)

2. **Add Tables**:
   - Click "Add Table" button
   - Choose shape (Round, Rectangle, Square)
   - Drag to position
   - Set capacity (number of seats)
   - Assign tier (VIP, Premium, General)

3. **Add Chairs**:
   - Click "Add Chair" button
   - Position around tables
   - Link to table (optional)

4. **Set Pricing**:
   - VIP Price: ₹5000
   - Premium Price: ₹3000
   - General Price: ₹1500

#### Step 3: Save Floor Plan
1. Give your floor plan a name
2. Click "Save"
3. Set as "Active" to enable seat selection

### Managing Exhibitors

#### View Exhibitor Applications
1. Go to event dashboard
2. Click "Exhibitors" tab
3. See all exhibitor applications

#### Approve Exhibitor
1. Review application details
2. Click "Approve" button
3. Booth number is auto-assigned
4. Exhibitor receives confirmation email

#### Set Pricing for Exhibitor
1. Click "Set Pricing" on approved exhibitor
2. Enter pricing details:
   - Base booth price
   - Electrical access fee
   - Display tables fee
   - Other charges
3. Add notes (payment terms, etc.)
4. Click "Finalize & Send Payment Link"
5. Exhibitor receives payment link via email

#### Process Refund
1. Find exhibitor with completed payment
2. Click "Refund" button
3. Enter refund amount
4. Enter reason for refund
5. Confirm refund
6. Exhibitor receives refund confirmation

### Event Day Check-In

#### Setup Check-In Station
1. Go to event dashboard
2. Click "Event Day" → "Check-In"
3. You'll see check-in interface

#### Using QR Code Scanner
1. Click "Scan QR Code" button
2. Allow camera access
3. Point camera at attendee's QR code
4. System automatically checks in
5. Success message appears

#### Manual Check-In
1. Use search bar to find attendee
2. Type name or email
3. Click "Check In" button next to attendee
4. Status updates to "Checked In"

#### View Check-In Stats
- Total Registrations: Shows total count
- Checked In: Shows how many checked in
- Real-time updates

### Viewing Analytics

#### Event Dashboard
1. Go to your event
2. View dashboard for quick stats:
   - Total registrations
   - Revenue
   - Check-in rate
   - Popular ticket types

#### Detailed Reports
1. Click "Analytics" tab
2. View charts and graphs:
   - Registration trends over time
   - Revenue by ticket type
   - Check-in statistics
   - Geographic distribution

#### Export Reports
1. Choose date range
2. Select metrics
3. Click "Export Report"
4. Download as PDF or Excel

## For Attendees

### Browsing Events

#### Find Events
1. Go to "Browse Events" or "Explore"
2. Use filters:
   - City
   - Date range
   - Event type
   - Price range
3. Click on event to view details

#### View Event Details
- Event description
- Date, time, venue
- Ticket prices
- Available seats (if applicable)
- Organizer information

### Registering for an Event

#### Step 1: Start Registration
1. Click "Register" or "Buy Tickets" button
2. Choose ticket type (if multiple options)

#### Step 2: Fill Registration Form
1. **Personal Information**:
   - First Name
   - Last Name
   - Email
   - Phone Number

2. **Additional Information** (if required):
   - Company
   - Dietary preferences
   - Special requirements
   - T-shirt size

#### Step 3: Select Seats (if applicable)
1. View interactive floor plan
2. Click on available seats (green)
3. Selected seats turn blue
4. Unavailable seats are red
5. Click "Continue" when done

#### Step 4: Apply Promo Code (optional)
1. Enter promo code in the field
2. Click "Apply"
3. Discount is applied to total

#### Step 5: Payment
1. Review order summary
2. Choose payment method:
   - Credit/Debit Card
   - UPI
   - Net Banking
3. Complete payment
4. Wait for confirmation

#### Step 6: Confirmation
1. You'll see success message
2. Check your email for:
   - Registration confirmation
   - QR code
   - Event details
3. Check SMS for check-in code
4. Check WhatsApp for QR code image

### Using Your QR Code

#### Receiving QR Code
- Email: Attached as image
- WhatsApp: Sent as message
- SMS: Check-in code sent

#### At the Event
1. Open email/WhatsApp with QR code
2. Show QR code to staff at check-in
3. Staff scans code
4. You're checked in!
5. Enjoy the event

#### If QR Code Doesn't Work
- Show check-in code from SMS
- Provide email address to staff
- Staff can manually check you in

### Managing Your Registrations

#### View Your Registrations
1. Log in to your account
2. Go to "My Registrations" or "My Tickets"
3. See all your upcoming events

#### Download Ticket
1. Click on registration
2. Click "Download Ticket" button
3. Save PDF to your device

#### Cancel Registration
1. Find your registration
2. Click "Cancel" button
3. Confirm cancellation
4. Refund processed (if applicable)

## For Exhibitors

### Applying for Booth Space

#### Step 1: Find Event
1. Browse events
2. Look for events accepting exhibitors
3. Click "Apply as Exhibitor"

#### Step 2: Fill Application
1. **Company Information**:
   - Company Name
   - Business Type
   - Website

2. **Contact Details**:
   - Contact Person Name
   - Email
   - Phone Number

3. **Booth Requirements**:
   - Preferred booth size (3x3, 3x6, etc.)
   - Booth type (Standard, Premium)
   - Electrical access needed (Yes/No)
   - Display tables needed (Yes/No)

4. **Additional Information**:
   - Products/Services to showcase
   - Special requirements
   - Notes for organizer

#### Step 3: Submit Application
1. Review all details
2. Click "Submit Application"
3. Wait for approval

### After Approval

#### Receive Confirmation
- Email with booth number
- Payment link
- Event details
- Setup instructions

#### Make Payment
1. Click payment link in email
2. Review pricing breakdown:
   - Base booth price
   - Electrical access fee
   - Display tables fee
   - GST
   - Total amount
3. Complete payment
4. Receive payment confirmation

### Booth Setup

#### Before Event Day
- Receive setup schedule
- Booth number and location
- Load-in/load-out times
- Parking information

#### On Event Day
1. Arrive during setup time
2. Find your booth number
3. Set up your display
4. Attend exhibitor briefing (if any)

## For Event Staff

### Check-In Process

#### Using Mobile Scanner
1. Open check-in page on mobile device
2. Click "Scan QR Code"
3. Point camera at attendee's QR code
4. Wait for confirmation
5. Direct attendee to enter

#### Manual Check-In
1. Ask attendee for email or name
2. Search in the system
3. Verify identity
4. Click "Check In"
5. Confirm success

#### Handling Issues

**QR Code Not Scanning**:
1. Ask for check-in code from SMS
2. Search by email
3. Manually check in

**Attendee Not Found**:
1. Verify event name
2. Check spelling of name/email
3. Contact organizer if needed

**Already Checked In**:
1. Inform attendee
2. Check check-in time
3. Allow entry if legitimate

### Managing On-Site Registrations

#### Walk-In Registration
1. Go to registration page
2. Fill attendee details
3. Collect payment (if required)
4. Generate QR code
5. Check in immediately

## FAQ

### General Questions

**Q: Is my data secure?**
A: Yes, we use industry-standard encryption and security practices.

**Q: Can I get a refund?**
A: Refund policy depends on the event organizer. Check event details.

**Q: How do I change my registration details?**
A: Contact the event organizer through the event page.

### For Organizers

**Q: How many events can I create?**
A: Unlimited events on all plans.

**Q: Can I customize registration forms?**
A: Yes, you can add custom fields in event settings.

**Q: How do I export attendee data?**
A: Go to Registrations tab and click Export button.

**Q: Can I send emails to all attendees?**
A: Yes, use the "Send Email" feature in Registrations tab.

### For Attendees

**Q: I didn't receive my QR code**
A: Check spam folder. Contact organizer if still not received.

**Q: Can I transfer my ticket to someone else?**
A: Depends on event settings. Contact organizer.

**Q: What if I lose my QR code?**
A: Log in to your account and download it again from My Registrations.

### Technical Issues

**Q: The website is slow**
A: Try clearing browser cache or using a different browser.

**Q: Payment failed but money was deducted**
A: Contact support with transaction ID. Refund processed within 7 days.

**Q: QR code scanner not working**
A: Ensure camera permissions are enabled in browser settings.
