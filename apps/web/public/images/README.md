# Event Category Images

This directory should contain image files for the event category cards displayed on the browse events page.

## Required Image Files

The following image files are required for the category cards:

1. `category-comedy.webp` - For comedy shows
2. `category-amusement.webp` - For amusement park events
3. `category-theatre.webp` - For theatre shows
4. `category-kids.webp` - For kids events
5. `category-music.webp` - For music shows
6. `category-sports.webp` - For sports events

## Image Specifications

- Format: WebP (preferred) or JPEG/PNG
- Resolution: 400x300px (minimum)
- Aspect ratio: 4:3 
- File size: Less than 100KB per image (optimized for web)

## Placeholder Implementation

Until actual images are added, the application will fall back to using colored gradients for each category card.
